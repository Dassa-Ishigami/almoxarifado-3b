# Tarefa de Requisitos

## Requisitos Funcionais do Sistema

Todas as funcionalidades que o sistema deve ter.

### [RF01] Cadastro de produto

Cadastrar o produto com os seguintes atributos:

- nome, marca e quantidade no estoque

### [RF02] Movimentação Entrada

- Deve Adicionar um valor inteiro à quantidade no **estoque**
  - O usuário deve ser capaz de adicionar esse valor

### [RF03] Movimentação Saída

- Deve Diminuir um valor inteiro à quantidade no *estoque*
  - O usuário deve ser capaz de remover ~~números~~ itens

### [RF04] Criação de conta para o Usuário

- O sistema deve ser capaz de permitir a criação de uma conta e registrá-la no banco de dados do sistema

### [RF05] Usuário deve fazer o Login e entrar na conta criada

- O Login para a conta do usuário deve ser um requisito funcional

## Requisitos Não-Funcionais

- Os limites do sistema e outras funções que não integram requisitos funcionais. São complementos para melhor funcionalidade do sistema.

## [RF01] Segurança do Banco de Dados

- O banco de dados deve ser acessado apenas pelo desenvolvedor do sistema
